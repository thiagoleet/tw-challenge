# TW Challenge (Java + Spring Boot + PostgresSQL)

## To run the project

### Requirements:
- Docker

The services can be run on the background with command:
```bash
docker-compose up -d
```

Stopping all the running containers is also simple with a single command:
```bash
docker-compose down
```

## Documentation
